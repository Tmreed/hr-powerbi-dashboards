# Accommodations Analytics Dashboard

## Overview
This project recreates an HR accommodations analytics dashboard built in Power BI.  
The dashboard tracks accommodation requests, approval timelines, and case status to help HR teams monitor ADA-related workflows.

All data used in this project is **synthetic and generated for demonstration purposes only**.

## Key Metrics
- Total accommodation requests
- Approved vs denied requests
- Average processing time
- Requests by department
- Request type distribution

## Data Model
The dataset simulates employee accommodation requests and includes:

- Employee ID
- Department
- Accommodation type
- Request status
- Request date
- Resolution date

## Tools Used
- Power BI
- Excel
- Synthetic HR datasets

## Purpose
This project demonstrates how HR teams can use analytics to monitor accommodation workflows, identify delays, and improve compliance reporting.
