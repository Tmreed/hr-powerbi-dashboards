\# HR Power BI Dashboards



This repository contains recreated HR analytics dashboards built in Power BI using synthetic data.



The dashboards are based on real-world HR analytics scenarios but do not contain any proprietary or confidential company information.



\## Projects



\### Accommodations Dashboard

Tracks accommodation requests, status, and processing timelines to help HR teams monitor ADA-related workflows.



\### Employee Relations Dashboard

Provides visibility into workplace complaints, investigations, and case resolution trends.



\### Immigration Dashboard

Tracks visa types, expiration timelines, and workforce immigration risk indicators.



\### Exit Analytics Dashboard

Analyzes exit interview trends and workforce turnover insights.
